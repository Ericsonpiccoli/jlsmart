Como Funciona a Instalação Técnica

Para garantir a melhor experiência na automação da sua casa, seguimos um processo estruturado:

Procedimentos para Fazer o Pedido:

  1. Consultoria Inicial: Entre em contato conosco para agendar uma consultoria onde avaliaremos suas necessidades.
  2. Aprovação do Projeto: Apresentamos um projeto técnico detalhado para sua aprovação.
  3. Pedido: Após a aprovação, você formaliza o pedido e escolhe as opções de pagamento.
  4. Agendamento da Instalação: Definimos uma data para a instalação técnica.

Instalação Técnica:

A instalação é realizada por nossa equipe especializada, que seguirá estas etapas:

  1. Análise do Local: Nossos técnicos farão uma análise do local para identificar a melhor forma de instalação dos dispositivos.
  2. Instalação dos Equipamentos: Os equipamentos serão instalados com o máximo cuidado para garantir o funcionamento ideal.
  3. Configuração e Testes: Após a instalação, realizamos a configuração dos dispositivos e testes para assegurar que tudo está funcionando perfeitamente.
  4. Treinamento: Oferecemos um treinamento para você e sua família sobre como utilizar os sistemas de automação.
  5. Acompanhamento Pós-Instalação: Disponibilizamos suporte técnico após a instalação para resolver qualquer dúvida ou problema.
